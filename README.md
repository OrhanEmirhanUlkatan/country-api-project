
# Project Title

Country Api Project


## Run Locally

Clone the project

```bash
  git clone https://github.com/OrhanEmirhanUlkatan/country-api-project.git
```

Go to the project directory

```bash
  cd countrt-api-project
```

Install dependencies

```bash
  npm install || yarn
```

Start the server

```bash
  npm start || yarn start
```
